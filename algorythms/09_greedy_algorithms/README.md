# Greedy Algorithms

## Definition

A greedy algorithm makes the best local choice at each step, hoping that these choices combine into a global optimum.

## Core ideas

- Local optimum vs global optimum
- Exchange arguments
- Sorting before greedy choice
- Interval reasoning

## Simple examples

- Activity selection
- Minimum number of arrows for balloons
- Jump game reasoning

## Use cases

- Scheduling
- Resource allocation
- Interval optimization

## Problems this topic solves

- Produces elegant and fast solutions when a greedy-choice property exists
- Avoids heavier dynamic programming when not needed

## What to practice

- Jump game
- Gas station
- Non-overlapping intervals
- Partition labels
