# Binary Search Trees

## Definition

A binary search tree (BST) is a binary tree where values in the left subtree are smaller than the node and values in the right subtree are larger.

## Core ideas

- Ordered tree property
- Search, insert, and delete
- Inorder traversal produces sorted output

## Simple examples

- Insert numbers and print them in sorted order
- Search for a value faster than scanning a list

## Use cases

- Ordered dictionaries
- Range queries
- Maintaining sorted dynamic data

## Problems this topic solves

- Supports efficient ordered operations
- Makes predecessor, successor, and range lookups practical

## What to practice

- Validate BST
- Lowest common ancestor in BST
- Search in BST
- Insert into BST
- Delete node in BST
