# Tree Algorithms

## Definition

Tree algorithms focus on traversal, search, decomposition, balancing logic, and dynamic reasoning over hierarchical structures.

## Core ideas

- DFS traversals
- BFS traversals
- Recursive divide and combine
- Lowest common ancestor
- Path and subtree calculations

## Simple examples

- Sum all node values
- Find maximum depth
- Compare two trees

## Use cases

- File systems
- Expression evaluation
- Organizational charts
- Search indexes

## Problems this topic solves

- Enables efficient reasoning over hierarchical data
- Supports path, ancestor, and aggregation queries

## What to practice

- Binary tree level order traversal
- Lowest common ancestor
- Path sum
- Serialize and deserialize binary tree
- Binary tree maximum path sum
