# Graph Algorithms

## Definition

Graph algorithms process networks of nodes and edges to answer questions about reachability, shortest paths, connectivity, and ordering.

## Core ideas

- BFS and DFS
- Topological sort
- Shortest paths
- Minimum spanning tree
- Union-find for connectivity

## Simple examples

- Find whether a path exists
- Compute shortest route
- Order tasks with dependencies

## Use cases

- Maps and navigation
- Dependency management
- Social networks
- Network routing

## Problems this topic solves

- Explains and solves relationship-driven systems
- Handles pathfinding and dependency ordering efficiently

## What to practice

- Course schedule
- Network delay time
- Number of islands
- Clone graph
- Accounts merge
