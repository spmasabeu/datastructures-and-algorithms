# Backtracking

## Definition

Backtracking explores possible choices step by step, and when a choice cannot lead to a valid solution it undoes that choice and tries another path.

## Core ideas

- Decision trees
- Choose, explore, unchoose
- Pruning
- Constraint satisfaction

## Simple examples

- Generate permutations
- Solve sudoku
- Find all subsets

## Use cases

- Combinatorial search
- Puzzle solving
- Constraint-based generation

## Problems this topic solves

- Enumerates valid solutions when brute force is necessary
- Cuts down wasted work through pruning

## What to practice

- Subsets
- Permutations
- Combination sum
- N-Queens
- Word search
