# Hash Tables and Sets

## Definition

A hash table stores key-value pairs using a hash function. A set stores unique values, often using the same hashing ideas.

## Core ideas

- Hash functions
- Collisions and collision handling
- Average-case constant time lookup
- Unique membership testing

## Simple examples

- Word frequency counter
- Duplicate detection
- Caching results

## Use cases

- Fast lookups
- Counting frequencies
- Indexing data
- Memoization

## Problems this topic solves

- Speeds up searches dramatically
- Tracks seen items efficiently
- Associates keys with useful metadata

## What to practice

- Two-sum
- Group anagrams
- Top k frequent elements
- Contains duplicate
- Longest consecutive sequence
