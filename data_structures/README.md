# Data Structures Roadmap

This folder is ordered from foundational to more advanced topics.

## Suggested order

1. `01_arrays_strings`
2. `02_linked_lists`
3. `03_stacks`
4. `04_queues_deques`
5. `05_hash_tables_sets`
6. `06_heaps_priority_queues`
7. `07_trees_binary_trees`
8. `08_binary_search_trees`
9. `09_tries_disjoint_sets`
10. `10_balanced_trees`
11. `11_graphs`
12. `12_advanced_structures`

## Why this sequence

It starts with the structures you will use constantly in interviews, competitive programming, and production systems, then moves into the ones that sharpen deeper problem-solving and systems thinking.
