# Two Pointers and Sliding Window

## Definition

Two pointers uses two moving indices to scan data efficiently. Sliding window maintains a dynamic subarray or substring while expanding and shrinking boundaries.

## Core ideas

- Opposite-direction pointers
- Same-direction pointers
- Fixed-size window
- Variable-size window

## Simple examples

- Reverse a string in place
- Find pair sums in sorted arrays
- Longest substring under a constraint

## Use cases

- String processing
- Array optimization
- Range-based counting problems

## Problems this topic solves

- Replaces nested loops with linear scans in many cases
- Efficiently tracks local constraints over contiguous ranges

## What to practice

- Valid palindrome
- Two sum II
- Container with most water
- Longest repeating character replacement
- Minimum size subarray sum
