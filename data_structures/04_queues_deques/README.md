# Queues and Deques

## Definition

A queue is a First In, First Out (FIFO) data structure. A deque allows insertion and removal from both ends.

## Core ideas

- Enqueue and dequeue
- FIFO ordering
- Front and rear pointers
- Double-ended operations in deques

## Simple examples

- Printer job scheduling
- Breadth-first search
- Sliding window maximum with deque

## Use cases

- Task scheduling
- Real-time buffering
- Graph traversal
- Window-based optimization

## Problems this topic solves

- Fair processing order
- Efficient front removal and back insertion
- Flexible window maintenance with deques

## What to practice

- Implement queue using stacks
- Number of recent calls
- Sliding window maximum
- Rotting oranges
