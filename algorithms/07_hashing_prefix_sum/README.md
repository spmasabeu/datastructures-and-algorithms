# Hashing and Prefix Sum

## Definition

Hashing enables fast lookup by key. Prefix sums precompute cumulative totals so range queries become fast.

## Core ideas

- Frequency maps
- Seen-set tracking
- Prefix sums and running totals
- Prefix modulo and hash map combinations

## Simple examples

- Count repeated values
- Compute sums of subarrays quickly
- Detect target subarray sums

## Use cases

- Frequency analysis
- Subarray problems
- Data deduplication
- Fast range statistics

## Problems this topic solves

- Replaces repeated scanning with cached information
- Makes many subarray sum problems near linear time

## What to practice

- Subarray sum equals k
- Ransom note
- Continuous subarray sum
- Product of array except self
