# Dynamic Programming

## Definition

Dynamic programming solves problems with overlapping subproblems and optimal substructure by storing intermediate results.

## Core ideas

- Memoization
- Tabulation
- State definition
- Transition formula
- Base cases

## Simple examples

- Fibonacci with caching
- Climbing stairs
- Coin change

## Use cases

- Optimization problems
- Counting paths or combinations
- Sequence alignment and matching

## Problems this topic solves

- Turns exponential recursion into polynomial-time solutions
- Finds optimal answers in structured search spaces

## What to practice

- House robber
- Longest increasing subsequence
- Coin change
- Longest common subsequence
- Edit distance
