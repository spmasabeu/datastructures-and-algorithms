# Divide and Conquer

## Definition

Divide and conquer breaks a problem into smaller subproblems, solves them independently, and combines the results.

## Core ideas

- Split
- Solve recursively
- Merge results
- Recurrence relations

## Simple examples

- Merge sort
- Quick sort
- Maximum subarray variants

## Use cases

- Efficient sorting
- Large-scale computations
- Recursive optimization

## Problems this topic solves

- Reduces complex tasks into manageable pieces
- Often improves performance significantly

## What to practice

- Merge sort
- Quick sort
- Count inversions
- Majority element
