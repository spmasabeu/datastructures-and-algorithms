# Tries and Disjoint Sets

## Definition

A trie stores strings by characters along paths in a tree. A disjoint set, also called union-find, keeps track of groups of connected elements.

## Core ideas

- Trie prefix traversal
- Fast prefix lookups
- Union and find operations
- Path compression and union by rank

## Simple examples

- Autocomplete with a trie
- Group connected computers with union-find

## Use cases

- Dictionaries and autocomplete
- Prefix search
- Connected components
- Cycle detection in undirected graphs

## Problems this topic solves

- Trie: fast prefix queries
- Disjoint set: fast connectivity checks and merges

## What to practice

- Implement trie
- Add and search word
- Number of connected components
- Redundant connection
