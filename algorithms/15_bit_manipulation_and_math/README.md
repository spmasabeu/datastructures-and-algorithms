# Bit Manipulation and Math

## Definition

Bit manipulation uses binary representation directly for compact and fast operations. Math algorithms handle numeric patterns, modular arithmetic, primes, and combinatorics.

## Core ideas

- Bitwise AND, OR, XOR, NOT
- Shifts
- Masks
- Modular arithmetic
- GCD, LCM, sieves, fast exponentiation

## Simple examples

- Check whether a number is a power of two
- Count set bits
- Compute greatest common divisor

## Use cases

- Performance-sensitive code
- Cryptography
- Number theory problems
- State compression in dynamic programming

## Problems this topic solves

- Encodes state compactly
- Speeds up many low-level operations
- Solves mathematical constraints cleanly

## What to practice

- Single number
- Counting bits
- Reverse bits
- Pow(x, n)
- Sieve of Eratosthenes
