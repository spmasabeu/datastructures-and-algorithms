# Arrays and Strings

## Definition

Arrays store elements in contiguous memory and allow fast access by index. Strings are ordered sequences of characters and are often treated like arrays with text-specific operations.

## Core ideas

- Index-based access
- Traversal and scanning
- In-place updates
- Fixed-size vs dynamic arrays
- String parsing and manipulation

## Simple examples

- Reverse an array
- Find the maximum value
- Check whether a string is a palindrome
- Count character frequency

## Use cases

- Storing sequential data
- Processing text input
- Building buffers and tables
- Foundation for many higher-level structures

## Problems this topic solves

- Fast lookup by position
- Efficient iteration over ordered data
- Text cleanup, parsing, and comparison

## What to practice

- Two-sum
- Remove duplicates
- Rotate array
- Anagram checks
- Longest substring without repeating characters
