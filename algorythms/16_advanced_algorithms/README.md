# Advanced Algorithms

## Definition

This topic covers deeper algorithmic tools used in harder interviews, competitive programming, and performance-critical systems.

## Core ideas

- Segment trees
- Binary indexed trees
- Dijkstra and A*
- Bellman-Ford and Floyd-Warshall
- Network flow
- Union-find optimization
- Advanced dynamic programming

## Simple examples

- Fast shortest path on weighted graphs
- Range update and query systems
- Max-flow matching problems

## Use cases

- Routing engines
- Scheduling systems
- Competitive programming
- High-scale optimization problems

## Problems this topic solves

- Handles constraints where simpler approaches time out
- Opens the door to elite-level problem solving

## What to practice

- Dijkstra shortest path
- Range query problems
- Maximum bipartite matching
- DP on trees or bitmasks
