# Trees and Binary Trees

## Definition

A tree is a hierarchical data structure made of nodes connected by edges. A binary tree is a tree where each node has at most two children.

## Core ideas

- Root, parent, child, leaf, depth, and height
- Recursive structure
- Traversals: preorder, inorder, postorder, level order

## Simple examples

- Represent folder hierarchies
- Evaluate expression trees
- Traverse nodes level by level

## Use cases

- Hierarchical data
- Recursive problem modeling
- Search and aggregation in structured data

## Problems this topic solves

- Represents nested relationships naturally
- Enables divide-and-conquer traversal strategies

## What to practice

- Maximum depth of binary tree
- Invert binary tree
- Level order traversal
- Same tree
- Diameter of binary tree
