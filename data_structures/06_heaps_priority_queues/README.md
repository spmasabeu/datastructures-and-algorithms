# Heaps and Priority Queues

## Definition

A heap is a tree-based structure that keeps the smallest or largest element at the top. A priority queue uses a heap to always remove the highest-priority item first.

## Core ideas

- Min-heap vs max-heap
- Insert and extract operations
- Partial ordering
- Efficient top-k processing

## Simple examples

- Find the smallest element quickly
- Merge sorted lists
- Schedule tasks by priority

## Use cases

- Task schedulers
- Streaming median and top-k problems
- Graph algorithms like Dijkstra

## Problems this topic solves

- Efficient access to highest or lowest priority items
- Better performance than sorting everything repeatedly

## What to practice

- Kth largest element
- Merge k sorted lists
- Top k frequent elements
- Task scheduler
