# Heap and Selection Algorithms

## Definition

These algorithms use heaps or partition-based methods to quickly find largest, smallest, or top-k elements without fully sorting all data.

## Core ideas

- Priority queues
- Quickselect
- Streaming top-k maintenance
- Median maintenance

## Simple examples

- Find kth largest value
- Track top trending items
- Merge multiple sorted streams

## Use cases

- Ranking systems
- Recommendation feeds
- Real-time analytics

## Problems this topic solves

- Avoids unnecessary full sorting
- Efficiently maintains partial order statistics

## What to practice

- Kth largest element in an array
- Top k frequent elements
- Find median from data stream
- Merge k sorted lists
