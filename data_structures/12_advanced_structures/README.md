# Advanced Structures

## Definition

This topic groups specialized structures used when basic arrays, lists, heaps, and trees are not enough for high-performance or highly structured problems.

## Core ideas

- Segment trees
- Fenwick trees
- Suffix arrays
- Sparse tables
- Skip lists
- LRU cache design

## Simple examples

- Range sum queries with updates
- Fast text indexing
- Ordered probabilistic search with skip lists

## Use cases

- Competitive programming
- Search engines
- Databases and caches
- Real-time range analytics

## Problems this topic solves

- Fast range queries
- Fast repeated updates
- Specialized indexing and caching challenges

## What to practice

- Range sum query mutable
- Design LRU cache
- Study segment tree and Fenwick tree implementations
