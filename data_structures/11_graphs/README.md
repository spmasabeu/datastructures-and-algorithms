# Graphs

## Definition

A graph is a collection of vertices and edges used to model relationships, connections, or transitions between entities.

## Core ideas

- Directed vs undirected
- Weighted vs unweighted
- Adjacency list vs adjacency matrix
- Traversal with BFS and DFS

## Simple examples

- Social network connections
- Road maps
- Dependency graphs

## Use cases

- Routing
- Recommendation systems
- Scheduling dependencies
- Network analysis

## Problems this topic solves

- Models relationships that are not purely linear or hierarchical
- Solves pathfinding, connectivity, and dependency questions
