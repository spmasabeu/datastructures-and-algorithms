# Searching and Binary Search

## Definition

Searching algorithms find a target value or position in data. Binary search repeatedly halves a sorted search space to locate the answer efficiently.

## Core ideas

- Linear search
- Binary search on arrays
- Binary search on answer space
- Boundary finding

## Simple examples

- Find a number in a sorted array
- Find the first occurrence of a target
- Search the minimum feasible value

## Use cases

- Databases
- Configuration tuning
- Threshold optimization problems

## Problems this topic solves

- Faster search in sorted or monotonic spaces
- Efficiently finds boundaries and feasibility limits

## What to practice

- Binary search
- Search insert position
- First bad version
- Find peak element
- Search in rotated sorted array
