# Sorting Foundations

## Definition

Sorting arranges data in a chosen order, usually ascending or descending, so later operations become easier or faster.

## Core ideas

- Comparison sorts
- Stable vs unstable sorting
- In-place vs extra-memory approaches
- Bubble, selection, and insertion sort as learning tools

## Simple examples

- Sort numbers
- Sort words alphabetically
- Sort records by score

## Use cases

- Preprocessing for binary search
- Ranking and reporting
- Grouping similar values

## Problems this topic solves

- Simplifies searching, deduplication, and range processing
- Builds intuition for more advanced sorts

## What to practice

- Implement insertion sort
- Compare stable and unstable sorting
- Sort colors
